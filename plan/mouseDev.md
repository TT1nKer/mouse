## 二、发射端（桌面基站）

* 类型：VCSEL或IR LED（850–940nm）
* 功能：把电能转为红外光束

* 遮挡检测：任何障碍物遮挡光束立即关断
* Class-1 眼安全标准：保证人眼暴露在安全功率以下

## 四、光学指向原理（文本描述）

1. 光源产生红外光，通过准直透镜变为平行或窄束光。
2. 振镜或微云台旋转或倾斜，改变光束出射角。
3. 光束经过扫描透镜（f-θ镜），角度转化为鼠标平面上的光斑位置。
4. 鼠标端PSD/相机测光斑偏移，反馈给控制器。
5. 控制器调整振镜/微云台角度，使光束持续精确指向鼠标。

**公式**：

* 振镜反射角：θ_spot ≈ 2 × δ_mirror
* 自由空间传播：x_spot ≈ D × θ_spot

* **带宽**：200–500 Hz

### 1.4 PSD/相机传感器

* **输入**：反射或散射光（光斑）
* **输出**：Δx, Δy 偏差信号（模拟电压或数字 I²C/ADC）
* **带宽**：>500 Hz
* **功能**：精确跟踪光斑位置

### 1.5 MCU / 控制器

* **输入**：

  * PSD/相机 Δx/Δy
  * 粗定位方向（BLE AoA/UWB）
  * 鼠标端电压/电流反馈
* **输出**：

  * 振镜/微云台角度命令
  * IR 发射功率调节
* **功能**：闭环指向控制 + 功率闭环 + 安全保护

---

## 2. 鼠标端（接收端）

### 2.1 PV 光伏阵列

* **输入**：IR 光束，光斑直径 5–10 mm
* **输出**：直流电

  * Vmp ≈ 3–3.5 V
  * Imp ≈ 50–100 mA
* **附加**：可选电压/电流反馈接口

### 2.2 MPPT 模块

* **输入**：PV 电压/电流
* **输出**：稳定功率供给储能缓冲
* **控制信号**：实时调节 PV 工作点
* **效率**：>90%
* **接口**：模拟或数字测量点可反馈 DC/DC 控制器

### 2.3 储能缓冲（超容 / 小电池）

* **输入**：MPPT 输出
* **输出**：给 DC/DC 稳压器
* **容量**：超容 0.1–0.5F，或 LiPo 50–200 mAh
* **功能**：遮挡瞬时支撑，负载尖峰缓冲

### 2.4 DC/DC 升降压稳压器

* **输入**：储能缓冲电压 2–5.5V
* **输出**：稳定 5V

  * 输出电流 ≥150 mA
  * 纹波 <50 mVpp
* **控制信号**：可接受外部电压/电流反馈调节输出

### 2.5 输出滤波器

* **输入**：DC/DC 输出
* **输出**：平滑稳压 5V，低纹波
* **组件**：LC/π 型滤波

### 2.6 Optional 电池并联（ORing）

* **输入**：DC/DC 或内置电池
* **输出**：5V 稳压供鼠标
* **功能**：遮挡或电流尖峰时无缝切换供电来源

## 3. 模块数据/电气流概述（文本描述）

1. **光能流**：IR 发射器 → 振镜/透镜 → 鼠标 PV → MPPT → 储能 → DC/DC → 鼠标负载
2. **控制信号流**：

   * PSD/相机 → MCU → 振镜/微云台
   * 鼠标端电压/电流 → MCU → 发射功率调节
3. **电源流**：

   * USB 5V → 发射端驱动器 + 控制器
   * 光能 → PV → 储能 → DC/DC → 5V 鼠标
4. **反馈环路**：

   * 角度反馈（光斑偏移）闭环振镜/微云台
   * 电压反馈闭环调节 IR 发射功率
