# 无线鼠标项目资金预算计算说明

## 一、成本计算依据

### （一）器件成本来源

**1. 主控芯片价格**
- nRF52840：查询Digi-Key、Mouser等分销商价格
- 1K+数量：约$6.5（45元）
- 10K+数量：约$5.8（40元）
- 价格基于2024年Q3市场报价

**2. 光学传感器**
- PMW3360：PixArt官方代理商报价
- 1K+数量：约$5（35元）
- 主要供应商：世强、骏龙科技

**3. 光伏电池**
- GaAs多结电池：参考Spectrolab、AZUR SPACE产品
- 8mm×6mm规格：约$12（80元）
- 效率>30%的高效电池价格较高

**4. 振镜系统**
- 双轴振镜：参考Thorlabs、Newport产品
- 小型化振镜+驱动器：约$25（180元）
- 响应频率500Hz级别

**5. VCSEL激光器**
- 850nm波长：参考Lumentum、Finisar产品
- 1W输出功率：约$17（120元）
- 工业级温度范围

### （二）制造成本估算

**1. PCB成本**
- 4层PCB（鼠标端）：30元/片（100片起）
- 6层PCB（发射端）：50元/片（100片起）
- 基于嘉立创、华强PCB报价

**2. SMT贴片成本**
- 鼠标端：约15元/片（包含在PCB成本中）
- 发射端：约25元/片（包含在PCB成本中）
- 基于深圳SMT代工厂报价

**3. 外壳注塑成本**
- 开模费：鼠标外壳8万，发射端外壳5万
- 单件成本：鼠标25元，发射端40元
- 基于1万套年产量摊销

## 三、销售预测计算

### （一）市场容量估算

**目标市场规模**：
- 国内高端鼠标市场：约50万台/年
- 我们目标占有率：第1年0.8%，第5年11.8%
- 对应销量：4000台 → 59000台

**价格策略**：
- 参考竞品定价：罗技G Pro（899元），雷蛇毒蝰（1299元）
- 我们的技术溢价：+30-50%
- 最终定价：1299-2199元

### （二）销量增长模型

**增长假设**：
- 第1年：市场导入期，主要是尝鲜用户
- 第2-3年：口碑传播，销量快速增长
- 第4-5年：市场成熟，增长趋缓

**具体计算**：
```
第1年：4000台（保守估计）
第2年：4000×4 = 16000台（口碑爆发）
第3年：16000×2 = 32000台（渠道扩张）
第4年：32000×1.44 = 46000台（市场渗透）
第5年：46000×1.28 = 59000台（增长放缓）
```

### （三）收入计算

**产品组合假设**：
- 标准版：专业版：电竞版 = 5:4:2（销量比）
- 平均售价：1299×0.45 + 1699×0.36 + 2199×0.18 = 1592元

**年收入计算**：
```
2024年：4000×1592 = 637万 ≈ 640万
2025年：16000×1600 = 2560万
2026年：32000×1600 = 5120万
2027年：46000×1600 = 7360万
2028年：59000×1600 = 9440万
```

## 十一、向老板要钱实用指南（重点看这章）

**实际场景**：
1. 你把 `report.md` 提交给老板作为项目方案
2. 老板可能拿去拉投资（融800-1200万种子轮）
3. **你现在要向老板申请：我们2人的工资 + 启动资金**

### 核心策略：申请 180-200万（工资+启动资金）

**钱的具体构成**：
- 2人工资（6个月）：18-25万
  - 每人15K/月 × 2人 × 6个月 = 18万
  - 社保公积金：6.3万
- 核心设备采购：25万
- 打样试产：30万
- 专利申请：15万
- 外包服务：10万
- 办公场地：0万（在家办公省钱）
- **总计：104万核心 + 20-30万预留 = 125-135万**

**给老板的直接说明**：
> "老板，report我已经交了。现在需要申请启动资金：
> - 我们2人6个月工资+社保：25万
> - 核心设备+打样试产+专利：75万

**你现在要做的**：
