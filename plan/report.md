# 无线鼠标无线供电项目商业计划书

## 一、项目概述

本项目开发一套无线鼠标供电系统，包含桌面发射基站和鼠标接收端。通过红外光束实现无线供电，解决无线鼠标频繁充电问题。

**产品组成**:
- 鼠标端：基于nRF52840芯片，集成光伏接收和无线通信
- 发射端：桌面基站，红外激光发射和光束跟踪系统

**技术指标**:
- 工作距离：30-80cm
- 供电功率：100mW持续输出
- 鼠标性能：1000Hz轮询率，16000 DPI，延迟<1ms
- 跟踪精度：光束跟踪误差<0.5mm

## 二、技术方案

### （一）无线供电系统设计

**1. 发射端（桌面基站）**

整体规格：尺寸120mm×80mm×40mm，功耗≤15W

核心器件配置：
- 红外光源：VCSEL激光器（波长850nm），输出功率1W
- 光束控制：双轴振镜（型号待定），响应频率500Hz
- 位置检测：PSD四象限光电传感器+小型相机模组
- 主控芯片：STM32F407VGT6（LQFP-100封装）
- 电源管理：AC-DC适配器输入，内部多路DC-DC转换

光学系统：
- 准直透镜：焦距25mm，光斑直径5-8mm@50cm距离
- 振镜扫描角度：±10°，覆盖桌面80cm×60cm区域
- 跟踪算法：PID控制+卡尔曼滤波，跟踪误差<0.5mm

安全保护：
- 遮挡检测：红外接收管监测反射光强
- 功率限制：符合IEC 60825-1 Class 1标准
- 温度保护：内置温度传感器，过热自动降功率

**2. 接收端（鼠标内部）**

整体规格：接收模块尺寸15mm×10mm×3mm

器件选择：
- 光伏阵列：GaAs多结电池，有效面积8mm×6mm
- MPPT芯片：BQ25570（QFN-16封装），效率>90%
- 储能器件：超级电容0.22F/5.5V + 备用锂电池50mAh
- 稳压芯片：TPS63000（QFN-10封装），输出5V/150mA

电源管理策略：
- 光照充足时：光伏直供+超容充电
- 短时遮挡：超容放电维持供电50-200ms
- 长时遮挡：切换至备用电池供电

遮挡场景与解决方案（LOS约束）：
- 视线遮挡不可避免：杯子/手掌/纸张等遮挡会导致入射功率瞬时下降。
- 瞬时遮挡容错：提升超容容量至0.33-0.47F并优化MPPT阈值，保障>300ms供电连续性。
- 多束/多基站冗余：可选配“双发射基站”或单基站多VCSEL子阵列交错扫描，降低单路径被完全遮挡的概率。
- 宽角低功率保底束：在扫描束之外，叠加一束大角度、Class 1安全限值内的“保底束”，维持低功率续航与链路心跳。
- 遮挡预测：融合鼠标IMU与振镜回波，采用卡尔曼预测路径，提前调整光路绕避。
- 混合供电预留：在鼠标内预留小线圈与整流接口，可兼容电磁传导（鼠标垫/桌垫）作长期遮挡场景下的备份供电。

### （二）鼠标硬件平台

**主控芯片**：Nordic nRF52840
- 封装：QFN-48（7mm×7mm）
- 内核：ARM Cortex-M4F @ 64MHz
- 无线：蓝牙5.0，支持HID协议
- 存储：1MB Flash + 256KB RAM

**光学传感器**：PixArt PMW3360
- 接口：SPI通信
- 分辨率：400-12000 DPI可调
- 跟踪速度：250 IPS
- 加速度：50G

**按键开关**：欧姆龙D2FC-F-7N微动开关
- 寿命：2000万次点击
- 触发力：75gf
- 响应时间：<0.2ms

**其他器件**：
- 滚轮编码器：阿尔卑斯EC11系列
- 指示LED：0603封装，多色可选
- 外壳材质：ABS+PC，磨砂处理

## 主要风险

### 技术风险
**无线供电效率问题**: 如果光电转换效率达不到预期，可能影响使用体验。
- 应对：准备多套技术方案，已有技术储备

**安全认证**: 激光器产品需要通过各种安全认证。
- 应对：提前联系认证机构，按标准设计

**成本控制**: 初期成本可能偏高。
- 应对：优化设计，寻找更多供应商

**视线（LOS）遮挡风险**: 光束供电需直达视线，手部/物体遮挡会造成功率中断。
- 应对：
  - 提升超容与电源调度，确保短时遮挡不断电
  - 增加多束与多基站冗余，提高链路可用性
  - 上报遮挡状态并智能降载，保障鼠标HID最低功耗运行
  - 预留/提供电磁传导备选（无线充电鼠标垫/桌垫）以覆盖长时遮挡使用场景
### 市场风险  
**大厂跟进**: 如果罗技、雷蛇等大厂也做类似产品。
- 应对：专利保护，快速迭代，建立技术壁垒

**用户接受度**: 新技术可能需要时间被接受。
- 应对：找KOL合作，让用户体验到好处

**价格敏感**: 如果定价太高可能影响销量。
- 应对：做多个价位的产品线

### 资金风险
**研发投入大**: 前期投入大，回报周期长。
- 应对：分阶段融资，控制现金流

**量产良品率**: 新技术量产可能有良品率问题。
- 应对：找有经验的代工厂合作
