# 激光无线供电方案详细技术文档
## Laser Wireless Power Transfer Solution - Technical Details

---

## 一、能量传输原理 (Energy Transfer Principle)

### 1.1 总体架构

```
[发射端]                    [传输路径]                    [接收端]
桌面基站 ────────── 红外激光光束 ────────── 鼠标接收模块
   │                                  │
   │                                  │
电→光转换                          光→电转换
(850nm VCSEL)                    (GaAs光伏电池)
```

### 1.2 能量传输流程

**步骤1：电能输入**
- 桌面基站通过AC-DC适配器从市电（110V/220V）获取电能
- 输入功率：≤15W
- 经过内部多路DC-DC转换，为激光器提供稳定驱动电压

**步骤2：激光产生（电→光转换）**
- **器件**：VCSEL激光器（垂直腔面发射激光器）
- **波长**：850nm（近红外，人眼不可见）
- **输出功率**：1W（激光器输出）
- **光束特性**：
  - 准直透镜聚焦，光斑直径5-8mm @ 50cm距离
  - 光束发散角：通过透镜控制在±10°以内

**步骤3：光束传输**
- **传输介质**：空气（无物理连接）
- **传输距离**：30-80cm
- **跟踪机制**：双轴振镜实时调整光束方向，跟踪鼠标位置
- **跟踪精度**：误差<0.5mm

**步骤4：光能接收（光→电转换）**
- **接收器件**：GaAs（砷化镓）多结光伏电池
- **有效接收面积**：8mm × 6mm = 48mm²
- **转换效率**：GaAs多结电池典型效率25-35%（在850nm波长下）

**步骤5：功率管理与输出**
- **MPPT芯片**（BQ25570）：最大功率点跟踪，效率>90%
- **储能**：超级电容0.22-0.47F/5.5V
- **稳压输出**：TPS63000，输出5V/150mA给鼠标电路

## 二、详细能量转换过程

### 2.1 发射端：电能→光能转换

#### VCSEL激光器工作原理
```
直流电源 → 激光驱动电路 → VCSEL芯片 → 850nm红外激光
  15W输入              ↑                 ↑
                  电流控制            光子发射
```

**技术参数**：
- **发射波长**：850nm（近红外波段）
  - 选择原因：GaAs电池在该波长下转换效率较高
  - 人眼不可见，避免视觉干扰
  - 符合IEC 60825-1 Class 1安全标准（低功率）
  
- **输出功率**：1W
  - 考虑到传输损耗和转换效率，最终鼠标端获得100mW

- **光束质量**：
  - 通过准直透镜（焦距25mm）形成平行光束
  - 光斑直径：5-8mm @ 50cm距离
  - 光束发散角：通过透镜控制在±10°

#### 光束控制系统
**双轴振镜（Galvanometer Scanner）**：
- **功能**：快速调整激光束方向，跟踪鼠标位置
- **响应频率**：500Hz
- **扫描角度**：±10°（可覆盖桌面80cm×60cm区域）
- **控制精度**：跟踪误差<0.5mm

**位置检测系统**：
- **PSD四象限光电传感器**：检测反射光，确定鼠标大致位置
- **小型相机模组**：辅助精确定位和识别
- **跟踪算法**：PID控制 + 卡尔曼滤波

### 2.2 传输路径：光能在空气中的传播

```
激光器 → 准直透镜 → 振镜反射 → 空气传输 → 到达鼠标表面
  │        │         │          │
  │    聚焦光束   角度控制   30-80cm距离
  │
1W光功率
```

**传播特性**：
- **大气吸收**：850nm波长在空气中传播损耗极小（<1% @ 1米）
- **光束扩散**：通过准直透镜控制，距离越远光斑越大
  - @ 30cm：光斑直径约3-5mm
  - @ 50cm：光斑直径约5-8mm
  - @ 80cm：光斑直径约8-12mm

**功率密度计算**：
- 假设光斑直径8mm（@ 50cm），面积 = π×(4mm)² ≈ 50mm²
- 1W功率密度 = 1W / 50mm² = 20mW/mm²
- 接收面积48mm²的理论接收功率 = 20mW/mm² × 48mm² = **960mW**
- **实际接收功率**：考虑对准误差、接收效率等，实际约**100-200mW**

### 2.3 接收端：光能→电能转换

#### GaAs多结光伏电池

**工作原理**：
```
红外激光(850nm) → 照射GaAs电池 → 产生电子-空穴对 → 外部电路电流
  光子能量         吸收光能          载流子分离        电能输出
```

**技术参数**：
- **材料**：GaAs（砷化镓）多结电池
  - 多结结构提高对特定波长的吸收效率
  - 在850nm波长下转换效率较高（25-35%）

- **接收面积**：8mm × 6mm = 48mm²
  - 尺寸限制：必须适合鼠标内部空间
  - 位置：通常位于鼠标顶部，面向桌面基站方向

- **输出特性**：
  - 开路电压：约2.0-3.0V（取决于光照强度）
  - 短路电流：约50-100mA（取决于光功率）
  - 最大功率点：通过MPPT芯片动态跟踪

#### 功率管理电路

**MPPT芯片（BQ25570）**：
```
光伏电池输出 → BQ25570 MPPT → 最大功率点跟踪 → 输出至储能/负载
   可变电压         动态阻抗匹配        优化提取        稳定输出
```

- **功能**：最大功率点跟踪（Maximum Power Point Tracking）
- **效率**：>90%
- **原理**：实时调整工作点，使光伏电池始终工作在最大功率输出状态
- **重要性**：光照强度变化或部分遮挡时，自动调整以维持最大功率提取

**储能系统**：
```
MPPT输出 → 超级电容充电 → 稳压电路 → 鼠标电路供电
          (0.22-0.47F)    (TPS63000)    (5V/150mA)
```

- **超级电容**：0.22-0.47F/5.5V
  - **作用**：缓冲瞬时功率波动，应对短时遮挡
  - **容量选择**：
    - 0.22F可维持约50-200ms
    - 0.33-0.47F可维持>300ms（应对遮挡场景）
  - **充放电特性**：快速充放电，适合瞬时遮挡场景

- **备用锂电池**：50mAh
  - **作用**：长期遮挡时（>数秒）的备用电源
  - **切换策略**：超级电容电压低于阈值时自动切换

**稳压输出（TPS63000）**：
- **输入**：2.5-5.5V（来自超级电容）
- **输出**：5V/150mA（稳定电压）
- **效率**：>90%
- **功能**：为鼠标主电路（nRF52840、传感器等）提供稳定供电

## 四、能量转换效率分析

### 4.1 各环节效率

| 环节 | 输入功率 | 输出功率 | 效率 | 说明 |
|------|---------|---------|------|------|
| AC-DC转换 | 15W | 12-13W | ~85% | 适配器效率 |
| 激光驱动 | 12W | ~1W光 | ~8% | 电→光转换效率低 |
| 光束传输 | 1W光 | ~0.9W光 | ~90% | 大气吸收、散射损失 |
| 光→电转换 | 0.9W光 | ~0.25W | ~28% | GaAs电池效率 |
| MPPT管理 | 0.25W | ~0.23W | ~92% | MPPT芯片效率 |
| 稳压输出 | 0.23W | ~0.20W | ~87% | 稳压芯片效率 |
| **总效率** | **15W** | **0.1W** | **~0.67%** | 系统整体效率 |

**注**：虽然系统总效率较低（<1%），但对于100mW的小功率需求，15W输入是可接受的。关键优势是：
- 无需物理连接
- 可在桌面范围内自由移动
- 持续供电，无需充电

### 4.2 功率预算

**鼠标功耗需求**：
- nRF52840主控：~10-20mW（活跃时）
- PMW3360传感器：~15-30mW
- 按键、滚轮等：~5-10mW
- **总需求**：约50-80mW（正常使用）
- **峰值需求**：~100mW（高负载）

**功率余量**：
- 设计输出：100mW持续供电
- 正常使用：50-80mW（余量20-50mW用于超级电容充电）
- 遮挡时：超级电容维持供电50-300ms

## 五、关键技术细节

### 5.1 光束跟踪系统

**工作流程**：
1. **位置检测**：PSD传感器 + 相机模组检测鼠标位置
2. **位置预测**：卡尔曼滤波算法预测鼠标运动轨迹
3. **光束调整**：PID控制器调整振镜角度
4. **实时跟踪**：500Hz响应频率，跟踪误差<0.5mm

**控制算法**：
```
目标位置 = 鼠标当前位置（PSD/相机检测）
预测位置 = 卡尔曼滤波（当前位置 + 速度 + 加速度）
误差 = 预测位置 - 当前光束位置
振镜角度 = PID控制器（误差）
```

### 5.2 遮挡检测与应对

**遮挡检测**：
- **方法**：红外接收管监测反射光强度
- **原理**：正常情况下有部分反射光，遮挡时反射光强度骤降
- **响应速度**：<1ms检测到遮挡

**应对策略**：
1. **瞬时遮挡（<300ms）**：超级电容放电维持供电
2. **预测遮挡**：融合鼠标IMU数据，提前调整光路绕避
3. **多束冗余**：可选配多VCSEL阵列，降低完全遮挡概率
4. **保底光束**：宽角低功率光束维持基本供电
5. **混合供电**：预留电磁传导接口作备份

### 5.3 安全保护机制

**IEC 60825-1 Class 1标准**：
- **激光安全等级**：Class 1（人眼安全）
- **功率限制**：确保单点曝光功率在安全限值内
- **自动关闭**：检测到异常遮挡或温度过高时自动降功率或关闭

**温度保护**：
- **温度传感器**：监测激光器温度
- **过热保护**：温度过高时自动降功率，防止损坏

## 六、系统优势与局限

### 6.1 优势

1. **高功率密度**：激光传输功率密度高，适合小面积接收
2. **长距离传输**：30-80cm距离，适合桌面应用
3. **精确控制**：可精确跟踪和定位
4. **无干扰**：光传输不受电磁干扰影响
5. **持续供电**：无需充电，持续使用

### 6.2 局限

1. **LOS约束**：必须视线直达，遮挡会中断供电
2. **系统复杂**：需要精确的光学系统和跟踪算法
3. **成本较高**：激光器、振镜、光学系统成本较高
4. **效率较低**：电→光→电转换效率总体较低
5. **安全要求**：需要符合激光安全标准
