## 二、多线圈系统架构

### 2.1 系统类型

#### 类型1：多发射线圈（单接收）

```
发射端：
┌─────────────────────────────┐
│  [Tx1]  [Tx2]  [Tx3]  [Tx4]  │  ← 多个发射线圈
│  [Tx5]  [Tx6]  [Tx7]  [Tx8]  │
└─────────────────────────────┘
            ↓ 磁场耦合
        [接收线圈Rx]
```

**特点**：
- ✅ 覆盖范围大
- ✅ 可以跟踪移动设备
- ✅ 提高功率传输能力

---

#### 类型2：单发射线圈（多接收）

```
        [发射线圈Tx]
            ↓
┌─────────────────────────────┐
│  [Rx1]  [Rx2]  [Rx3]  [Rx4]  │  ← 多个接收线圈
│  [Rx5]  [Rx6]  [Rx7]  [Rx8]  │
└─────────────────────────────┘
```

**特点**：
- ✅ 同时为多个设备充电
- ✅ 负载均衡
- ⚠️ 耦合系数可能下降

---

#### 类型3：混合系统（部分发射+部分接收）⭐您问的

```
混合线圈阵列：
┌─────────────────────────────┐
│  [Tx]  [Rx]  [Tx]  [Rx]      │  ← 混合配置
│  [Rx]  [Tx]  [Rx]  [Tx]      │
└─────────────────────────────┘
```

**特点**：
- ✅ 灵活：可以切换功能
- ✅ 双向功率传输
- ⚠️ 系统复杂
- ⚠️ 需要智能控制

---

### 2.2 混合系统的实现方式

#### 方式1：可切换线圈

```
每个线圈可以切换：
- 发射模式（Tx）
- 接收模式（Rx）
- 关闭模式（Off）

控制方式：
- 根据需求动态切换
- 或固定配置
```

#### 方式2：固定角色分配

```
部分线圈固定为发射：
[Tx1] [Tx2] [Tx3]

部分线圈固定为接收：
[Rx1] [Rx2] [Rx3]

优点：简单
缺点：灵活性低
```

## 三、效率分析

### 3.1 单线圈系统（基准）

**效率公式**：
```
η_single = k² × Q₁ × Q₂ / (1 + k² × Q₁ × Q₂)
```

**典型参数**：
- k = 0.1-0.3（1米距离）
- Q = 100-200
- 效率：40-60%

---

### 3.2 多发射线圈系统

#### 情况A：同时激活所有发射线圈

**优点**：
- ✅ 提高总功率
- ✅ 磁场叠加，可能提高耦合

**缺点**：
- ❌ **线圈间耦合**（相互干扰）
- ❌ **效率可能下降**
- ❌ 功率分配不均

**效率分析**：

**问题1：线圈间耦合**
```
发射线圈之间：
M_Tx1-Tx2 ≠ 0  （相互耦合）

这会导致：
1. 功率在发射线圈间分流
2. 谐振频率偏移
3. 效率下降
```

**效率估算**：
```
η_multi_Tx = η_single × α

其中 α = 0.6-0.9（取决于线圈间距和配置）

如果线圈间距 < 线圈直径：
α ≈ 0.6-0.7（效率下降30-40%）

如果线圈间距 > 2×线圈直径：
α ≈ 0.8-0.9（效率下降10-20%）
```

**结论**：多发射线圈同时工作，**效率会下降10-40%**

---

#### 情况B：选择性激活（推荐）

**方法**：
- 根据接收端位置，只激活最近的1-2个发射线圈
- 其他线圈关闭

**优点**：
- ✅ 避免线圈间耦合
- 效率接近单线圈系统

**效率**：
```
η_selective ≈ η_single × 0.85-0.95

（控制电路有少量损耗）
```

**这是推荐的方法！**

---

### 3.3 多接收线圈系统

#### 情况A：并联接收线圈

```
发射线圈 Tx
    ↓
┌─────────┬─────────┐
│  Rx1    │  Rx2    │  ← 并联
│         │         │
└─────────┴─────────┘
```

**效率分析**：

**优点**：
- ✅ 提高总功率接收能力
- ✅ 增加接收面积

**缺点**：
- ⚠️ 接收线圈间可能耦合
- ⚠️ 需要功率分配

**效率**：
```
η_multi_Rx = η_single × β

β ≈ 0.8-0.95（取决于接收线圈配置）

如果接收线圈距离近：
β ≈ 0.8-0.85（效率下降15-20%）

如果接收线圈距离远：
β ≈ 0.9-0.95（效率下降5-10%）
```

---

#### 情况B：串联接收线圈

```
发射线圈 Tx
    ↓
Rx1 -- Rx2 -- Rx3（串联）
```

**效率分析**：
- ⚠️ 电压叠加，但电流受限
- 效率可能更低：η ≈ 0.7-0.85 × η_single

---

### 3.4 混合系统（部分发射+部分接收）⭐

#### 关键问题

**1. 发射线圈与接收线圈的耦合**

```
理想情况（希望）：
Tx1 ←→ Rx1  （强耦合）
Tx2 ←→ Rx2  （强耦合）

实际问题：
Tx1 ←→ Rx1  （目标耦合）
Tx1 ←→ Rx2  （干扰耦合）
Tx2 ←→ Rx1  （干扰耦合）
Tx2 ←→ Rx2  （目标耦合）

还有：
Tx1 ←→ Tx2  （发射线圈间耦合）
Rx1 ←→ Rx2  （接收线圈间耦合）
```

**2. 效率下降原因**

**原因1：线圈间交叉耦合**
```
多个耦合路径同时存在：
- 目标路径：Tx1 → Rx1
- 干扰路径：Tx1 → Rx2, Tx2 → Rx1

干扰路径：
- 消耗功率（能量分流）
- 降低效率
```

**原因2：谐振频率偏移**
```
多个线圈耦合 → 谐振频率改变
→ 需要重新调谐
```

**原因3：功率分配不均**
```
多个发射线圈：
- 功率分配可能不均
- 某些路径效率低
```

---

#### 效率估算

**简化模型**（2个发射+2个接收）：

**理想情况**（无交叉耦合）：
```
η_ideal = η_single_Tx1-Rx1 × P_Tx1/(P_total) 
        + η_single_Tx2-Rx2 × P_Tx2/(P_total)
```

**实际情况**（有交叉耦合）：
```
η_actual = η_ideal × γ

其中 γ = 0.5-0.8（取决于线圈配置）

如果线圈间距小（<线圈直径）：
γ ≈ 0.5-0.6（效率下降40-50%）

如果线圈间距大（>2×线圈直径）：
γ ≈ 0.7-0.8（效率下降20-30%）
```

## 四、优化策略

### 4.1 线圈布局优化

#### 方法1：增加线圈间距

**原则**：
```
线圈间距 > 2 × 线圈直径

好处：
- 减少线圈间耦合
- 提高效率

代价：
- 占用更大空间
- 可能降低覆盖范围
```

#### 方法2：错位布局

```
避免发射和接收线圈直接对准：
[Tx]    [Rx]    [Tx]    [Rx]
  ↓      ↑
 错位布局，减少交叉耦合
```

#### 方法3：选择性激活

**动态控制**：
- 只激活与接收端最近/最匹配的线圈
- 关闭其他线圈
- 减少干扰

---

### 4.2 频率分离

**方法**：
- 不同发射线圈使用不同频率
- 或使用频率分复用（FDM）
- 减少相互干扰

**缺点**：
- 系统复杂
- 成本增加

---

### 4.3 补偿网络优化

**方法**：
- 针对多线圈系统优化补偿网络
- 补偿线圈间耦合的影响
- 调整谐振频率

---

## 五、实际效率预测

### 5.1 场景1：2个发射+2个接收

**配置**：
```
[Tx1]  [Tx2]
  ↓      ↓
[Rx1]  [Rx2]
```

**效率预测**：
```
线圈间距 = 1.5×线圈直径：

目标耦合：Tx1-Rx1, Tx2-Rx2
效率：η ≈ 35-45%（相比单线圈50%，下降10-15%）

如果选择性激活（只激活匹配的Tx-Rx对）：
效率：η ≈ 45-50%（接近单线圈）
```

---

### 5.2 场景2：3×3阵列（混合配置）

**配置**：
```
[Tx] [Rx] [Tx]
[Rx] [Tx] [Rx]
[Tx] [Rx] [Tx]
```

**效率预测**：
```
线圈间距 = 1×线圈直径：

效率：η ≈ 20-30%（大幅下降）

原因：
- 大量交叉耦合
- 谐振频率偏移
- 功率分配不均
```

**优化后（增加间距，选择性激活）**：
```
线圈间距 = 2×线圈直径：

效率：η ≈ 35-45%（仍低于单线圈）
```

---

### 5.3 场景3：相控阵系统（推荐）

**配置**：
```
所有线圈都是发射线圈
通过相位控制形成定向波束
```

**优势**：
- ✅ 方向可控，减少无效耦合
- ✅ 效率可能高于混合系统
- ✅ 可以同时服务多个接收端

**效率预测**：
```
η_phased_array ≈ 45-60%（可能高于混合系统）

原因：
- 通过相位控制减少干扰
- 方向性更好
- 功率更集中
```

## 六、推荐方案

### 6.1 如果目标是多设备充电

**推荐：多发射线圈阵列（选择性激活）**

```
发射端：
多个发射线圈，根据接收端位置选择性激活

接收端：
每个设备一个接收线圈

优点：
- 效率高（接近单线圈）
- 可以同时服务多设备
- 灵活

效率：η ≈ 40-55%
```

---

### 6.2 如果目标是双向功率传输

**推荐：可切换线圈系统**

```
线圈可以切换：
- 发射模式
- 接收模式

动态切换，根据需求

优点：
- 灵活
- 支持能量回收

效率：η ≈ 40-50%（切换有损耗）
```

---

### 6.3 如果目标是提高功率和覆盖

**推荐：相控阵系统**

```
所有线圈都是发射
通过相位控制形成定向波束

优点：
- 方向可控
- 效率较高
- 覆盖范围大

效率：η ≈ 45-60%
```

## 七、结论

### 7.1 混合系统的效率

**关键结论**：

1. **效率会下降**
   - 相比单线圈系统，多线圈混合系统效率会**下降20-50%**
   - 原因：线圈间交叉耦合、谐振频率偏移、功率分配不均

2. **影响因素**
   - **线圈间距**：间距越大，效率越高（但需要更大空间）
   - **线圈配置**：如何布局发射和接收线圈
   - **控制策略**：选择性激活 vs 同时激活

3. **实际效率范围**
   - **紧凑布局**（间距<线圈直径）：η ≈ 20-35%
   - **宽松布局**（间距>2×线圈直径）：η ≈ 35-45%
   - **选择性激活**：η ≈ 40-50%

---

### 7.2 与单线圈对比

| 系统类型 | 典型效率 | 优点 | 缺点 |
|---------|---------|------|------|
| **单线圈** | 40-60% | 效率高，简单 | 覆盖范围小 |
| **多发射（同时）** | 25-40% | 功率大 | 效率低，干扰大 |
| **多发射（选择）** | 40-55% | 灵活，覆盖大 | 系统复杂 |
| **多接收** | 35-50% | 多设备 | 效率略低 |
| **混合系统** | 25-45% | 灵活 | 效率低，复杂 |
| **相控阵** | 45-60% | 方向可控，效率高 | 系统复杂，成本高 |

---

### 7.3 建议

**如果目标是最高效率**：
- ✅ 使用**单线圈系统**或**选择性激活的多发射系统**

**如果目标是灵活性**：
- ✅ 使用**相控阵系统**（效率较高）
- ⚠️ 避免混合系统（效率较低）

**如果必须使用混合系统**：
- ✅ 增加线圈间距（>2×直径）
- ✅ 优化线圈布局（减少交叉耦合）
- ✅ 选择性激活（只激活匹配的Tx-Rx对）
- ✅ 优化补偿网络
