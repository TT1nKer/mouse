# 谐振耦合无线充电可行性验证实验手册
## Resonant Inductive Coupling Wireless Power Transfer - Feasibility Test Lab Manual

---

## 一、实验目的

### 1.1 实验目标

**主要目标**：
验证谐振耦合无线充电技术在**中距离（0.5-1米）**和**中等功率（50-100W）**下的可行性，为后续产品原型开发提供技术验证。

**验证指标**：
1. ✅ **传输效率**：在0.5-1米距离下达到**≥40%**效率
2. ✅ **功率传输**：能够稳定传输**50-100W**功率
3. ✅ **谐振频率匹配**：发射端和接收端谐振频率误差**<5%**
4. ✅ **系统稳定性**：连续运行30分钟无故障

**参考标准**：
- 论文验证：arXiv:2411.17505 - 109.7W @ 1米，效率47.14%
- 目标：达到或接近论文水平

### 2.3 关键参数设计

#### 设计目标参数

| 参数 | 目标值 | 说明 |
|------|--------|------|
| **工作频率** | 85 kHz | Qi标准频率，成熟技术 |
| **发射线圈直径** | 40 cm | 平衡耦合和尺寸 |
| **接收线圈直径** | 25 cm | 相对发射线圈约60% |
| **工作距离** | 0.5-1 m | 主要测试距离 |
| **传输功率** | 50-100 W | 中等功率验证 |
| **目标效率** | ≥40% | 在1米距离下 |

#### 线圈设计参数

**发射线圈（L₁）**：
- **直径**：D₁ = 40 cm
- **匝数**：N₁ = 12 匝
- **线材**：Litz线，线径约2-3 mm
- **预期电感**：L₁ ≈ 50-60 µH
- **品质因数**：Q₁ ≥ 100（目标）

**接收线圈（L₂）**：
- **直径**：D₂ = 25 cm
- **匝数**：N₂ = 10 匝
- **线材**：Litz线，线径约2 mm
- **预期电感**：L₂ ≈ 20-30 µH
- **品质因数**：Q₂ ≥ 100（目标）

#### 谐振电容计算

**谐振频率**：f₀ = 85 kHz

**发射端电容**：
```
C₁ = 1 / (4π² × f₀² × L₁)
   = 1 / (4π² × (85×10³)² × 55×10⁻⁶)
   ≈ 63 nF
```

**接收端电容**：
```
C₂ = 1 / (4π² × f₀² × L₂)
   = 1 / (4π² × (85×10³)² × 25×10⁻⁶)
   ≈ 140 nF
```

**电容选择**：
- **类型**：C0G/NP0陶瓷电容（低损耗）
- **电压**：≥100V（留余量）
- **精度**：±5%
- **组合**：可用多个电容并联达到目标值

## 六、实验操作步骤

### 6.1 准备阶段

#### 步骤1：器件准备和检查

**任务清单**：
- [ ] 检查所有元器件：线圈、电容、MOSFET、二极管
- [ ] 使用LCR表测量线圈电感和Q值
- [ ] 记录实际参数值
- [ ] 计算所需谐振电容值
- [ ] 选择匹配的谐振电容

**操作方法**：
```
1. LCR表测量线圈：
   - 频率设置为85 kHz
   - 测量L₁, L₂, Q₁, Q₂
   - 记录数据

2. 计算谐振电容：
   C₁ = 1/(4π²×f₀²×L₁)
   C₂ = 1/(4π²×f₀²×L₂)

3. 选择电容：
   - 使用C0G/NP0陶瓷电容
   - 可并联多个达到目标值
   - 验证电容值（LCR表测量）
```

---

#### 步骤2：电路搭建

**发射端搭建**：
```
顺序：
1. 在面包板/PCB上搭建全桥逆变器
   - 安装MOSFET（Q1-Q4）
   - 安装驱动电路（MOSFET驱动IC）
   - 连接DC电源（24V）

2. 安装补偿网络和发射线圈
   - 串联谐振电容C₁和发射线圈L₁
   - 连接到逆变器输出

3. 连接测量点
   - 电压测量：示波器探头
   - 电流测量：电流探头或分流器

4. 检查连接
   - 检查所有连接是否正确
   - 检查是否有短路
```

**接收端搭建**：
```
顺序：
1. 安装补偿网络和接收线圈
   - 串联谐振电容C₂和接收线圈L₂

2. 安装整流电路
   - 全桥整流（4个肖特基二极管）
   - 滤波电容（100-470 µF）

3. 连接负载
   - 电子负载（可调电阻）或固定电阻

4. 连接测量点
   - 电压测量：万用表/示波器
   - 电流测量：万用表/电流探头
```

---

#### 步骤3：线圈安装和定位

**安装方法**：
```
1. 使用非金属支架固定线圈
   - 发射线圈：水平放置
   - 接收线圈：水平放置，与发射线圈平行

2. 距离设置
   - 初始距离：0.5米
   - 确保线圈同轴对齐
   - 使用测量工具确认距离

3. 标记位置
   - 标记不同测试距离的位置
   - 0.5m, 0.7m, 1.0m

4. 检查环境
   - 移除附近的金属物体
   - 确保测试区域无干扰
```

---

### 6.2 测试阶段

#### 步骤4：谐振频率测量和调谐

**4.1 测量谐振频率**

**方法A：网络分析仪（精确）**
```
1. 断开负载，只测量发射端
2. 网络分析仪连接到发射线圈
3. 频率扫描：80-90 kHz
4. 找到谐振频率（阻抗最小点）

6. 同样方法测量接收端
7. 记录：f₂（接收端谐振频率）

8. 计算频率误差：
   Δf = |f₁ - f₂|
   误差率 = Δf / f₀ × 100%
```

**方法B：信号发生器+示波器（简单）**
```
1. 信号发生器连接到发射线圈
2. 示波器监测发射线圈电压

5. 记录频率

6. 同样方法测量接收端
7. 检查频率匹配
```

**如果频率不匹配**：
```
方法1：微调电容

- 或更换不同值的电容

方法2：调整线圈
- 微调线圈匝数（如果可调）
- 或改变线圈间距

目标：使f₁ ≈ f₂ ≈ 85 kHz
容差：Δf < 5 kHz
```

## 八、安全注意事项

### 8.1 电气安全

**高压警告**：
- 发射线圈电压可能达到**100-200V**（谐振时）
- 接收线圈开路电压可能达到**50-100V**
- **操作时注意安全，避免触电**

**安全措施**：
1. ✅ 使用隔离变压器（如需要）
2. ✅ 测试前检查所有连接
3. ✅ 避免触摸高压部分
4. ✅ 使用绝缘工具
5. ✅ 接地良好

---

### 8.2 热安全

**发热源**：
- MOSFET开关损耗
- 线圈电阻损耗
- 二极管整流损耗

**安全措施**：
1. ✅ 监测温度（<60°C表面温度）
2. ✅ 使用散热器（MOSFET）
3. ✅ 确保通风良好
4. ✅ 避免长时间过载

---

### 8.3 电磁安全

**注意事项**：
- 强磁场可能影响附近电子设备
- 可能干扰无线通信（85 kHz影响较小）
- 移除附近的金属物体（避免涡流损耗）

## 九、故障排查指南

### 9.1 常见问题

**问题1：无法达到谐振**

**症状**：
- 接收端无电压或电压很低
- 效率极低（<10%）

**可能原因**：
1. 谐振频率不匹配（发射和接收频率不同）
2. 电容值错误
3. 线圈连接错误

**解决方法**：
1. 使用网络分析仪精确测量谐振频率
2. 重新计算和选择电容
3. 检查电路连接

---

**问题2：效率低于预期**

**症状**：
- 效率 <30%（1米距离）

**可能原因**：
1. 线圈Q值太低
2. 耦合系数太低（距离太远或线圈未对齐）
3. 补偿网络不合适
4. 开关损耗大

**解决方法**：
1. 使用Litz线提高Q值
2. 减小距离或优化线圈位置
3. 尝试不同补偿网络（S-S → LCC）
4. 使用高效率开关器件（SiC MOSFET）

---

**问题3：功率传输不稳定**

**症状**：
- 功率波动大
- 效率波动大

**可能原因**：
1. 谐振频率漂移（温度影响）
2. 负载变化
3. 电源不稳定

**解决方法**：
1. 温度补偿
2. 稳定负载
3. 使用稳压电源

## 十一、附录

### 11.1 计算公式

**谐振频率**：
```
f₀ = 1 / (2π × √(L × C))
```

**效率（简化）**：
```
η = P_out / P_in × 100%
```

**效率（理论）**：
```
η = k² × Q₁ × Q₂ / (1 + k² × Q₁ × Q₂)
```

**耦合系数（估算）**：
```
k ≈ √(η / (Q₁ × Q₂))
```

**品质因数**：
```
Q = ωL / R = 2πfL / R
```
