# 无线充电系统测试清单 / Wireless Charging System Test Checklist

## For Friend - Complete Testing Guide

---

## 📋 测试前准备 (Pre-Testing Setup)

### 必需设备清单 (Required Equipment List)

**基础测量设备 (Basic Measurement Equipment)**:

- [ ] LCR表 (LCR Meter) - 测量电感和Q值
- [ ] 示波器 (Oscilloscope) - 至少100 MHz带宽
- [ ] 万用表 (Multimeter) - 电压和电流测量
- [ ] 功率计 (Power Meter) 或 电压表+电流表组合
- [ ] **USB电流表或USB功率计** 🔥必需 - 监测USB输入功率和电流
- [ ] 电子负载 (Electronic Load) - 可编程负载

**高级设备（可选但推荐）(Advanced Equipment - Optional but Recommended)**:

- [ ] 网络分析仪 (Network Analyzer) - 精确频率测量
- [ ] 频谱分析仪 (Spectrum Analyzer) - EMI测试
- [ ] 热成像仪或热电偶 (Thermal Imager or Thermocouple) - 温度测量
- [ ] 电流探头 (Current Probe) - 电流波形测量
- [ ] 信号发生器 (Signal Generator) - 独立测试用

---

## 🔬 阶段一：预设计验证测试 (Phase 1: Pre-Design Validation)

### ✅ Test 1.1: 发射线圈参数测试 (Transmitter Coil Parameters)

**目标**: 验证发射线圈L₁的电感值和Q值

**步骤**:

1. [ ] 使用LCR表在谐振频率f₀附近（13.56 MHz或85 kHz）测量L₁
2. [ ] 记录测量值：L₁_measured = ______ µH
3. [ ] 比较：计算值 ≈ 5 µH，测量值应在 4.5-5.5 µH 范围内
4. [ ] 测量Q值（使用LCR表的Q功能）
5. [ ] 记录Q值：Q₁ = ______ （目标 > 100）

**通过标准**:

- [ ] L₁ = 5 µH ± 10% (4.5-5.5 µH)
- [ ] Q₁ > 100
- [ ] 记录测量值和误差

---

### ✅ Test 1.2: 接收线圈参数测试 (Receiver Coil Parameters)

**目标**: 验证接收线圈L₂的电感值和Q值

**步骤**:

1. [ ] 使用LCR表在谐振频率f₀附近测量L₂
2. [ ] 记录测量值：L₂_measured = ______ µH
3. [ ] 比较：计算值 ≈ 1-2 µH，测量值应在范围内
4. [ ] 测量Q值
5. [ ] 记录Q值：Q₂ = ______ （目标 > 100）

**通过标准**:

- [ ] L₂ = 1-2 µH ± 10%
- [ ] Q₂ > 100
- [ ] 记录测量值和误差

---

### ✅ Test 1.3: 谐振频率验证 (Resonant Frequency Verification)

**目标**: 验证实际谐振频率与设计值匹配

**步骤**:

1. [ ] 连接谐振电容C₁到发射线圈L₁
2. [ ] 使用网络分析仪（或LCR表）扫频
3. [ ] 找到阻抗最小的频率 = ______ MHz/kHz
4. [ ] 比较：设计值 f₀ = 13.56 MHz（或85 kHz）
5. [ ] 计算误差：误差 = |f_measured - f_design| / f_design × 100% = ______ %

**通过标准**:

- [ ] 频率误差 < ±1%
- [ ] 记录实际谐振频率

**重复接收端**:

- [ ] 连接C₂到L₂，重复上述步骤
- [ ] 记录接收端谐振频率 = ______

---

### ✅ Test 1.4: 补偿网络频率响应 (Compensation Network Frequency Response)

**设备**: 网络分析仪

**步骤**:

1. [ ] 连接完整LCC补偿网络（发射端）
2. [ ] 使用网络分析仪测量S21（传输系数）
3. [ ] 验证在f₀处有峰值响应
4. [ ] 记录峰值频率 = ______
5. [ ] 测量3dB带宽 = ______
6. [ ] 计算：带宽百分比 = 带宽 / f₀ × 100% = ______ %

**通过标准**:

- [ ] 峰值频率 = f₀ ± 1%
- [ ] 3dB带宽 < 1% f₀

---

### ✅ Test 1.5: 阻抗匹配验证 (Impedance Matching Verification)

**步骤**:

1. [ ] 使用网络分析仪测量发射端输入阻抗 Z_in = ______ Ω
2. [ ] 使用网络分析仪测量接收端输出阻抗 Z_out = ______ Ω
3. [ ] 计算匹配度

**通过标准**:

- [ ] Z_in ≈ 50Ω（与源阻抗匹配）
- [ ] Z_out ≈ R_load（与负载匹配）

---

## ⚡ 阶段二：发射端独立测试 (Phase 2: Transmitter Standalone Tests)

### ✅ Test 2.1: 逆变器开关频率验证 (Inverter Switching Frequency)

**设备**: 示波器

**步骤**:

1. [ ] 连接示波器探头到MOSFET栅极（Q1或Q2）
2. [ ] 上电，观察PWM波形
3. [ ] 测量开关频率 f_switch = ______
4. [ ] 测量占空比 = ______ %（目标：50% ± 2%）
5. [ ] 测量死区时间 = ______ ns（目标：100-500 ns）

**通过标准**:

- [ ] f_switch = f₀ ± 1%
- [ ] 占空比 = 50% ± 2%
- [ ] 死区时间 = 100-500 ns

---

### ✅ Test 2.2: 开关波形质量 (Switching Waveform Quality)

**设备**: 示波器（高带宽）

**测量项**:

1. [ ] 上升时间 t_r = ______ ns（目标 < 50 ns）
2. [ ] 下降时间 t_f = ______ ns（目标 < 50 ns）
3. [ ] 振铃幅度 = ______ V，占峰值百分比 = ______ %（目标 < 20%）
4. [ ] 过冲幅度 = ______ V，占峰值百分比 = ______ %（目标 < 10%）
5. [ ] 拍照保存波形图

**通过标准**:

- [ ] 所有指标在目标范围内
- [ ] 波形清晰，无明显振铃或过冲

---

### ✅ Test 2.3: 开关损耗测量 (Switching Loss Measurement)

**设备**: 示波器 + 电流探头

**步骤**:

1. [ ] 使用示波器同时测量V_DS和I_DS波形
2. [ ] 记录电压和电流峰值
3. [ ] 计算或使用功率分析仪测量开关损耗
4. [ ] P_switch = ______ W
5. [ ] P_total = ______ W（总功率）
6. [ ] 计算百分比：P_switch / P_total × 100% = ______ %

**通过标准**:

- [ ] P_switch < 5% P_total

---

### ✅ Test 2.4: 导通损耗测量 (Conduction Loss Measurement)

**步骤**:

1. [ ] 测量线圈RMS电流 I_rms = ______ A
2. [ ] MOSFET导通电阻 R_DS(on) = ______ Ω（查 datasheet）
3. [ ] 计算：P_cond = I_rms² × R_DS(on) × 2 = ______ W
4. [ ] 计算百分比：P_cond / P_total × 100% = ______ %

**通过标准**:

- [ ] P_cond < 10% P_total
- [ ] 总损耗（开关+导通）< 15% P_total

---

### ✅ Test 2.5: 线圈电流测试 (Coil Current Test)

**设备**: 电流探头 + 示波器

**步骤**:

1. [ ] 使用电流探头测量发射线圈电流 I_coil
2. [ ] 记录RMS电流：I_coil_RMS = ______ A
3. [ ] 观察波形，检查是否有异常尖峰
4. [ ] 验证电流在安全范围内（< 2A，根据设计）

**通过标准**:

- [ ] I_coil_RMS < 2A（根据设计调整）
- [ ] 无异常尖峰

---

### ✅ Test 2.6: 谐振状态验证 (Resonant State Verification)

**步骤**:

1. [ ] 同时测量线圈电压和电流
2. [ ] 使用示波器的相位测量功能
3. [ ] 测量相位差 = ______ 度（目标 < 10°）
4. [ ] 验证电流幅度在谐振频率下最大

**通过标准**:

- [ ] 相位差 < 10°（接近同相）
- [ ] 电流幅度符合设计

---

### ✅ Test 2.7: 过流保护测试 (Overcurrent Protection Test)

**步骤**:

1. [ ] 系统正常工作，记录正常电流 I_nominal = ______ A
2. [ ] 逐渐增加负载（使用电子负载）
3. [ ] 观察电流上升
4. [ ] 记录保护触发时的电流 I_protect = ______ A
5. [ ] 验证系统关闭或限流
6. [ ] 计算：I_protect应该 ≈ 1.5 × I_nominal

**通过标准**:

- [ ] I_protect = 1.5 × I_nominal ± 5%
- [ ] 保护功能正常工作
- [ ] 无器件损坏

---

### ✅ Test 2.8: 温度保护测试 (Temperature Protection Test)

**设备**: 热风枪 + 温度计

**步骤**:

1. [ ] 测量系统初始温度 T_initial = ______ °C
2. [ ] 使用热风枪加热系统（小心操作）
3. [ ] 观察温度上升
4. [ ] 记录保护触发时的温度 T_protect = ______ °C
5. [ ] 验证系统关闭
6. [ ] 等待冷却后重新测试

**通过标准**:

- [ ] T_protect = 60-70°C
- [ ] 保护功能正常工作

---

### ✅ Test 2.9: 故障检测测试 (Fault Detection Test)

**测试A - 空载测试**:

1. [ ] 系统正常工作
2. [ ] 断开接收端（空载）
3. [ ] 观察：系统应检测到故障并保护
4. [ ] 记录反应时间 = ______ ms

**测试B - 短路测试**:

1. [ ] 系统正常工作
2. [ ] 短路接收端输出
3. [ ] 观察：系统应检测到故障并保护
4. [ ] 记录反应时间 = ______ ms
5. [ ] 检查：无器件损坏

**通过标准**:

- [ ] 空载保护正常
- [ ] 短路保护正常
- [ ] 无器件损坏

---

### ✅ Test 2.10: USB功率控制测试 (USB Power Control Test) 🔥重要

**目标**: 验证USB输入功率控制在限制范围内（≤1W, ≤200mA）

**设备**: USB电流表 + 电压表 或 USB功率计

**关键约束**:
- USB 2.0: 5V @ 500mA = 2.5W 最大
- USB 3.0: 5V @ 900mA = 4.5W 最大
- **设计目标**: ≤ 1W (5V @ 200mA) - 安全限制

**测试A - USB输入电流限制**:

1. [ ] 连接USB电流表到USB输入
2. [ ] 系统正常工作，接收端连接
3. [ ] 测量USB输入电流 I_USB = ______ mA
4. [ ] 测量USB输入电压 V_USB = ______ V
5. [ ] 计算输入功率 P_in = V_USB × I_USB = ______ W
6. [ ] 记录在不同负载下的USB电流：
   - 空载（无接收端）：I_USB = ______ mA, P_in = ______ W
   - 轻载（50mW输出）：I_USB = ______ mA, P_in = ______ W
   - 正常负载（100mW输出）：I_USB = ______ mA, P_in = ______ W
   - 重载（150mW输出）：I_USB = ______ mA, P_in = ______ W

**通过标准**:
- [ ] I_USB ≤ 200mA 在所有情况下
- [ ] P_in ≤ 1W 在所有情况下

---

**测试B - USB过流保护测试**:

1. [ ] 系统正常工作，正常负载
2. [ ] 逐渐增加负载（使用电子负载）
3. [ ] 监测USB电流上升
4. [ ] 记录保护触发点：
   - USB电流触发值 I_trip = ______ mA（目标：220mA）
   - 此时系统反应：______（应降低功率或关闭）
5. [ ] 验证系统在电流限制范围内工作
6. [ ] 等待电流下降到安全值
7. [ ] 验证系统自动恢复

**通过标准**:
- [ ] 保护触发点在 220mA ± 10mA
- [ ] 保护功能正常工作
- [ ] 系统能自动恢复

---

**测试C - 待机功耗测试**:

1. [ ] 无接收端时（待机模式）
2. [ ] 测量USB电流 I_standby = ______ mA
3. [ ] 计算待机功耗 P_standby = V_USB × I_standby = ______ mW
4. [ ] 验证接收端检测功能正常（低功耗待机）

**通过标准**:
- [ ] I_standby < 20mA（待机功耗 < 100mW）
- [ ] 接收端检测功能正常

---

**测试D - 功率自适应测试**:

1. [ ] 初始状态：无接收端，记录USB电流 = ______ mA
2. [ ] 放置接收端，观察功率变化
3. [ ] 记录接收端连接后USB电流 = ______ mA
4. [ ] 记录功率上升时间 = ______ ms
5. [ ] 移除接收端，观察功率下降
6. [ ] 记录功率下降时间 = ______ ms
7. [ ] 验证功率控制响应速度

**通过标准**:
- [ ] 接收端连接后功率上升 < 500ms
- [ ] 接收端断开后功率下降 < 500ms
- [ ] USB电流始终 < 200mA

---

**测试E - 不同USB端口兼容性**:

**测试不同USB端口**:
- [ ] USB 2.0端口：电流 = ______ mA, 功率 = ______ W, 功能正常？
- [ ] USB 3.0端口：电流 = ______ mA, 功率 = ______ W, 功能正常？
- [ ] USB-C端口（非PD）：电流 = ______ mA, 功率 = ______ W, 功能正常？
- [ ] 笔记本电脑USB端口：电流 = ______ mA, 功率 = ______ W, 功能正常？
- [ ] 台式机前面板USB：电流 = ______ mA, 功率 = ______ W, 功能正常？

**通过标准**:
- [ ] 所有USB端口下电流 < 200mA
- [ ] 所有端口功能正常
- [ ] 无兼容性问题

## 📡 阶段三：接收端独立测试 (Phase 3: Receiver Standalone Tests)

### ✅ Test 3.1: 整流效率测试 (Rectifier Efficiency Test)

**设备**: 信号发生器 + 示波器 + 功率计

**步骤**:

1. [ ] 使用信号发生器产生交流信号（频率 = f₀）
2. [ ] 连接整流电路
3. [ ] 测量输入功率 P_in = ______ W
4. [ ] 测量输出功率 P_out = ______ W
5. [ ] 计算效率：η_rectifier = P_out / P_in × 100% = ______ %

**通过标准**:

- [ ] η_rectifier > 85%

---

### ✅ Test 3.2: 输出电压纹波测试 (Output Voltage Ripple Test)

**设备**: 示波器

**步骤**:

1. [ ] 测量整流输出直流电压 V_DC = ______ V
2. [ ] 使用示波器AC耦合模式测量交流纹波
3. [ ] 测量纹波峰峰值 V_ripple_pp = ______ V（滤波前）
4. [ ] 计算纹波百分比 = V_ripple_pp / V_DC × 100% = ______ %
5. [ ] 连接滤波电容后，重复测量纹波（滤波后）

**通过标准**:

- [ ] 滤波前：纹波 < 10% V_out
- [ ] 滤波后：纹波 < 1% V_out

---

### ✅ Test 3.3: 二极管压降测试 (Diode Forward Voltage Drop)

**设备**: 万用表

**步骤**:

1. [ ] 在不同电流下测量每个二极管的V_f
2. [ ] 记录二极管D1: V_f1 = ______ V @ I = ______ mA
3. [ ] 记录二极管D2: V_f2 = ______ V @ I = ______ mA
4. [ ] 记录二极管D3: V_f3 = ______ V @ I = ______ mA
5. [ ] 记录二极管D4: V_f4 = ______ V @ I = ______ mA
6. [ ] 计算平均值 V_f_avg = ______ V

**通过标准**:

- [ ] V_f < 0.3V @ I_rated
- [ ] 所有二极管压降接近

---

### ✅ Test 3.4: DC-DC转换器效率测试 (DC-DC Converter Efficiency Test)

**步骤**:

1. [ ] 轻载测试（10%负载）：

    - 输入功率 P_in_10% = ______ W
    - 输出功率 P_out_10% = ______ W
    - 效率 η_10% = ______ %
2. [ ] 中载测试（50%负载）：

    - 输入功率 P_in_50% = ______ W
    - 输出功率 P_out_50% = ______ W
    - 效率 η_50% = ______ %
3. [ ] 重载测试（100%负载）：

    - 输入功率 P_in_100% = ______ W
    - 输出功率 P_out_100% = ______ W
    - 效率 η_100% = ______ %

**通过标准**:

- [ ] 所有负载下效率 > 85%

---

### ✅ Test 3.5: DC-DC输出电压精度 (DC-DC Output Voltage Accuracy)

**步骤**:

1. [ ] 设置目标输出电压 V_set = 3.7V
2. [ ] 在不同负载下测量实际输出电压：
    - 轻载（10%）：V_out_10% = ______ V
    - 中载（50%）：V_out_50% = ______ V
    - 重载（100%）：V_out_100% = ______ V
3. [ ] 计算误差百分比（相对于3.7V）

**通过标准**:

- [ ] V_out = V_set ± 2%
- [ ] 所有负载下稳定

---

### ✅ Test 3.6: DC-DC动态响应测试 (DC-DC Dynamic Response Test)

**设备**: 示波器 + 电子负载

**步骤**:

1. [ ] 设置负载为10%
2. [ ] 使用电子负载快速切换到90%负载（阶跃变化）
3. [ ] 使用示波器捕获输出电压波形
4. [ ] 测量恢复时间 = ______ µs（目标 < 100 µs）
5. [ ] 测量过冲幅度 = ______ V，百分比 = ______ %（目标 < 5%）
6. [ ] 拍照保存波形图

**通过标准**:

- [ ] 恢复时间 < 100 µs
- [ ] 过冲 < 5% V_out
- [ ] 无振荡

---

## 🔗 阶段四：系统集成测试 (Phase 4: System Integration Tests)

### ✅ Test 4.1: 距离-效率关系测试 (Distance vs Efficiency Test) ⚡包含USB功率监测

**设备**: 功率计 + USB电流表 + 可调距离支架

**测试表格**:

| 距离 (cm) | USB电流 (mA) | P_in_USB (W) | P_in_Tx (W) | P_out (W) | 效率 (%) | USB限制检查 | 通过/失败 | 备注 |
| --------- | ------------ | ------------ | ----------- | --------- | -------- | ----------- | --------- | ---- |
| 0.5       |              |              |             |           |          |             |           |      |
| 1.0       |              |              |             |           |          |             |           |      |
| 2.0       |              |              |             |           |          |             |           |      |
| 5.0       |              |              |             |           |          |             |           |      |

**关键测量项**:
- **P_in_USB**: USB输入功率（必须 ≤ 1W）
- **P_in_Tx**: 发射端实际功率（Buck转换器后）
- **P_out**: 接收端输出功率
- **效率**: η = P_out / P_in_USB × 100%

**步骤**:

1. [ ] 连接USB电流表，监测USB输入电流
2. [ ] 将接收端放在0.5cm距离
3. [ ] 同时测量：
   - USB输入电流 I_USB = ______ mA
   - USB输入电压 V_USB = ______ V（通常5V）
   - USB输入功率 P_in_USB = ______ W
   - 发射端功率 P_in_Tx = ______ W（Buck后）
   - 输出功率 P_out = ______ W
4. [ ] 计算系统效率 η = P_out / P_in_USB × 100% = ______ %
5. [ ] **验证**: I_USB ≤ 200mA, P_in_USB ≤ 1W
6. [ ] 重复1.0cm, 2.0cm, 5.0cm距离
7. [ ] 绘制距离-效率曲线图
8. [ ] 绘制距离-USB电流曲线图

**通过标准**:

- [ ] 距离0-2cm：η > 70%

## 📝 注意事项 (Important Notes)

   - 确保设备精度满足要求
   - **必须准备USB电流表或USB功率计**

   - **所有测试中必须监测USB输入电流**
   - USB电流必须始终 ≤ 200mA

   - 如果超过限制，立即停止测试并检查
   - 这是系统安全的关键要求！

   - **特别记录USB电流和功率数据**
